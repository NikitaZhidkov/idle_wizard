# Game Characteristics

## Base Stats

| Stat | Description |
|------|-------------|
| **ATK** | Attack damage dealt to enemies |
| **DEF** | Defense - reduces incoming damage |
| **HP** | Health points |
| **Crit** | Critical hit chance (%) |

## Bonus Stats (from rewards)

| Stat | Description |
|------|-------------|
| **Crit Damage** | Critical hit damage multiplier |
| **Lifesteal** | Heal % of damage dealt |
| **Dodge** | Chance to avoid enemy attacks |
| **Thorns** | Reflect damage back to attacker |
| **Regen** | HP regeneration per turn |
| **Execute Damage** | Bonus damage to enemies below 30% HP |
| **Spell Power** | Increases spell effectiveness |
| **Death Saves** | Survive a killing blow (revive at 30% HP) |
| **Double Attack** | Attack twice per turn |
| **Gold Bonus** | Extra gold from kills |
| **XP Bonus** | Extra experience gain |

## Resources

| Resource | Description |
|----------|-------------|
| **Gold** | Currency earned from defeating enemies |
| **Gems** | Premium currency |
| **Level** | Character level (increases base stats) |
| **Floor** | Current dungeon floor |

## Stat Growth Per Level

- ATK: +2 per level
- DEF: +1 per level
- HP: +10 per level
